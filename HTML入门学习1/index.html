常用的表章节的标签有哪些，分别是什么意思**

章节标签是用来表示文章或书籍的层级，以下是标签的具体分类

  • 标题 h1 ~ h6

    <h1> 文章标题 </h1>

  • 章节 section

  • 段落 p

    <p>我真的超级棒</p>

  • 文章 article

  • 头部 header

    一般是用 header 标签来加网页的置顶广告 <header>这是一个好的广告位</header>

  • 脚部 footer

    一般用 footer 标签来加自己网站的版权声明 <footer>Amazeseek版权所有</footer>

  • 主要内容 main

    表示内容里主要的部分，通过使用 main 能够让文章的结构更加具有层次感。

  • 旁支内容 aside

    表示内容里次要的部分，一般在 aside 标签里添加网站的导航。

  • 划分 div

    通过 div 标签将 main 中的内容和 aside 中的内容包含起来，作为一个整体，这样层次感更加突出。

全局属性有哪些

全局属性是指所有标签都具备的属性，主要有以下几类

  • class

    给标签分个类，其实就是给标签做个标记。

  • contenteditable

    用户可以直接编辑页面上的文字，可以用来做一个编辑器。

  • hidden

    作用是快速的的让一个东西看不见。

  • id

    1. 一个作用是 在 CSS 中加样式
    2. 另一个作用是在 JS 中 是可以直接通过 id 获取元素的，但是这些元素中有些是禁忌的，需要注意。

    id 与 class 到底用谁？

    如果元素是全页面唯一的就用 id ，反之用 class ，还有就是不到万不得已千万不要用 id， 因为 id 的全局唯一性是不能保证的，同时也不会报错。

  • style

    每个元素都可以写一个 style 属性，注意 style属性的优先级要比 CSS 高。如果 CSS HTML 与 JS 的属性同时存在，则最终呈现效果以 JS 为准。

  • tabindex

    页面所有的可交互的属性都可用 tab 完成，tabindex 为正数时，以顺序的方式分别访问，不必是连续的正数。tabindex = 0 时，表示最后访问，若为 - 1 则表示不访问，tab找不到。

常用的内容标签有哪些，分别是什么意思

  • ol + li 有序列表
  • ul + li 无序列表
  • dl + dt +dd 对象描述列表
  • pre 如果想要保留空格 就用 pre
  • hr 分割线
  • br 换行
  • a 超链接
  • em 强调 表示语气很重要 强调句子成分
  • strong 重要 表示本身很重要
  • code 代码标签
  • quote 引用
  • blockquote 换行引用
