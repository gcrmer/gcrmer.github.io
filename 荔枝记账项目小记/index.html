LocalStorage 用法小总结

1.什么是 LocalStorage
localStorage, 是一个用来做本地持久化存储的 Web Api，它以键值对的形式存储数据。

2.LocalStorage 常用的 API 以及作用
clear: 清空 localStorage 上存储的数据
getItem:读取数据
hasOwnProperty:检查 localStorage 上是否保存了变量 x，需要传入 x
key:读取第 i 个数据的名字或称为键值(从 0 开始计数)
length: localStorage 存储变量的个数
propertyIsEnumerable: 用来检测属性是否属于某个对象的
setItem: 存储数据
removeItem: 删除某个具体变量
valueOf: 获取所有存储的数据
toLocaleString: 将数组转为本地字符串

让 React 应用支持 sass，遇到的问题以及解决的方法

1.需要 node-sass 但是下载它非常慢，而且本地编译也非常的慢 2.我想用 dart-sass 代替 node-sass，但是 React 不支持 dart-sass 3.在使用 google 搜索发现了一个解决办法 4.就是 npm6.9 支持一个新的功能，它叫做 package alias
5.yarn add node-sass@npm:dart-sass 即可实现，其实就表面上安装的是 node-sass，但实际上安装的是 dart-sass 6.完美解决了 dart-sass 代替 node-sass 的问题
