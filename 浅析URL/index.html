浅析URL

2021/08/26

本文于 342 天之前发表，文中内容可能已经过时。

URL 的构成与作用

URL 由 协议 + 域名或者 IP + 端口号+路径+ 查询字符串+锚点 几部分构成

协议：https 或者 http 作用是规定请求的格式与响应的格式

域名或者IP：作用是定位设备 封装数据报文以及同其他设备交流

端口号：作用是找到相应的服务

路径：请求不同的页面

查询字符串：查询同一个页面下 不同的内容

锚点：查询同一个内容下 不同的位置（注意锚点不支持中文且不会上传到服务器）

域名的含义与分类

域名是对 IP 的别称，它分为三类：顶级域名 二级域名 三级域名

如 cn 是顶级域名

tengchunlei.cn 是二级域名

www.tengchunlei.cn 是三级域名

DNS 的作用

DNS 的作用是将域名和 IP 对应起来，例如当我们在浏览器中输入 tengchunlei.cn 的时候，浏览器会向服务商提供的 DNS 服务器询问 tengchunlei.cn 对应的 IP 是多少，服务商会回答一个 IP ，然后浏览器会向对应的 IP 的 80/443 端口发送请求，内容是查看 tengchunlei.cn的首页。

nslookup 命令如何使用

nslookup 用于查询域名对应的 IP

如：nslookup google.com

IP 的作用 以及 ping 命令的使用

IP 的作用是定位设备 封装数据报文以及同其他设备交流

如果我们想知道一个域名的 IP 地址就可以通过 ping 命令

如： ping tengchunlei.cn
