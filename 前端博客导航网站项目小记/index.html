媒体查询的用法总结

Media Queries 是响应式设计的核心。

Media Queries 能在不同的条件下使用不同的样式，使页面在不同的终端设备下达到不同的渲染效果。

媒体特性是通过 min/max 来表示大于等于或小于做为逻辑判断。

Media Queries 在实际项目中常用的方式:

  1. 最大宽度 max-width

max-width 是媒体特性中最常用的一个特性，其意思是指媒体类型小于或等于指定的宽度时，样式生效。如:

上面表示的是：当页面宽度小于或等于 500px 时,searchForm 的颜色变成红色。

2.最小宽度 min-width
min-width 指的是媒体类型大于或等于指定宽度时，样式生效。

当页面宽度大于或等于 500px 时，globalHeader 上下间距设置为 60px;

3.多个特性使用

Media Queries 可以使用关键词”and”将多个特性结合在一起。也就是说，一个 Media Query 中可以包含 0 到多个表达式，表达式又可以包含 0 到多个关键字，以及一种媒体类型。

当页面宽度在 500px~900px 之间时，body 的背景色渲染为“#f5f5f5”。
