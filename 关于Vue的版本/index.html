2022/03/24

Vue 两种版本的区别

Vue 完整版:
有 complier,视图写在 HTML 里或者写在 template 选项，引入方式上从 cdn 引入时文件名为 vue.js，以@vue/cli 方式引入时，需要额外配置，以 webpack 方式引入时需要配置 alias。

Vue 非完整版:
无 complier，视图写在 render 函数里，用 h 来创建标签，引入方式上从 cdn 引入时文件名为 vue.runtime.js，以@vue/cli 方式引入时无需配置，默认使用这个版本，以 webpack 方式引入时无需配置，默认使用这个版本。
