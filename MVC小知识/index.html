MVC小知识

2022/03/19

关于模块化的理解？

模块化其实就是用一套规则将复杂且可读性不高的程序进行切割分块，不同的块之间保持各自的独立性，目的就是避免变量污染，命名冲突，提高代码的复用率与可维护性。

MVC 是做什么的？

M - Model 数据模型 处理数据

V - View 视图 负责所有的 UI 界面，当数据模型发生变化，视图也会更新。

C - Controller 控制器 定义用户界面对用户输入的相应方式，起到不同层面间的组织作用，用于控制应用程序的流程。

EventBus 的用处？

EventBus 是事件总线框架，可以用来简化组件之间通信和数据传输。它基于发布/订阅模式，实现业务代码解耦，提高开发效率。

EventBus 的 API

on 监听事件
trigger 触发事件
off 取消监听

表驱动编程的用处？

表驱动法就是一种编程模式(scheme)——从表里面查找信息而不使用逻辑语句(if 和 case)。事实上，凡是能通过逻辑语句来选择的事物，都可以通过查表来选择。对简单的情况而言，使用逻辑语句更为容易和直白。但随着逻辑链的越来越复杂，查表法也就愈发显得更具吸引力。表驱动编程能够将逻辑与数据进行分离，在我们面对条件表达式过多的情况下，可以使用表去查询，以便我们写出简洁优雅的代码。
