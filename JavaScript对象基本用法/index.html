JavaScript对象基本用法

2021/09/04

声明对象的两种语法

let obj = { ‘name’:’AmazeseeK’ ,’age’ :26 }

let obj = new Object({ ‘name’:’AmazeseeK’ })

如何删除对象的属性

delete obj.xxx 或delete obj[‘xxx’]

如何查看对象的属性

查看自身所有属性：Object.keys（obj）

查看自身 + 共有属性：console dir(obj)

查看属性的两种方法：

中括号语法：obj[‘key’]

点语法：obj.key

读对象的属性时，如果使用 [ ] 语法，那么 JavaScript 会先求 [ ] 中表达式的值，注意区分表达式是变量还是常量。
如果使用点语法，那么点后面一定是 string 常量。

如何修改或增加对象的属性

直接赋值：

let obj ={name: ‘AmazeseeK’}

obj.name = ‘AmazeseeK’

obj[‘name’] =’AmazeseeK’

批量赋值：

Object.assign(obj , {age: 26,gender:’man’})

‘name’ in object 和 obj.hasOwnProperty(‘name’)的区别

‘name’ in object ：能确定 object 中是否包含’name’属性

obj.hasOwnProperty(‘name’)：能确定’ name’属性是自身属性还是共有属性
