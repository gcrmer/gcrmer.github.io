jQuery 的链式操作是怎样的

jQuery 在选中网页元素以后，可以对它进行一系列操作，同时又可以将所有操作连接在一起，以链条的方式写出来。

链式操作的原理是每一步的 jQuery 操作返回的都是一个 jQuery 对象，所以不同的操作可以连在一起。

jQuery 如何移动元素

jQuery 提供两组方法，操作元素在网页中的位置移动。假设我们选中一个 div 元素，此时要将它移动到 p 元素后面，我们分别用以下两种方法实现：

方法一：使用 .insertAfter() 把 div 元素移动到 p 元素后面

方法二：使用 .after() 把 p 元素加到div元素前面

两种方法实现效果一样，但是有一个重大的差别就是返回的元素不一样，方法一返回的是 div 元素，方法二返回的是 p 元素。
