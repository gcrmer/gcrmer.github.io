JavaScript数据类型

2021/09/03

数据类型

数字 number
字符串 string
符号 symbol
布尔 bool
空 null
空 undefined
对象 object
bigint

数字:
特殊值 正 0 和负 0 都等于 0
无穷大 Infinity 、+Infinity、 -Infinity
无法表示的数字 NaN

字符串:
单引号 ‘AmazeseeK’
双引号 “AmazeseeK”
反引号 ``

转义:
\n 表示 换行
\r 表示 回车
\t 表示 tab 制表符

布尔 boolean：
只有 true 和 false 两个值
通过运算符得到 bool 值
否定运算 ！value
相等运算 2==3 、2!=3 、6===7 、 6!==7
比较运算 5>6、 5>=6 、 5<6 、 5<=6

if 语句常用来判断真假

当 value 是 bool 值时很容易判断真假
但是当 value 不是 bool 值时就很难判断真假,为了解决这个问题,采用 falsy 值的方法,falsy 就是相当于 false 但又不是 false 的值，它们分别是 undefined、 null 、0 、NaN 、’’

变量的声明方式

三种声明方式：
var a=3
let a=3
const a=3

区别： var 为过时不好用的方式
let 为新的更合理的方式
const 声明时必须赋值，且赋值后不能再更改

let 声明规则:
遵循块作用域，即使用范围不能超出{}
不能重复申明
可以赋值也可以不赋值
必须先声明在使用
全局声明 let 变量 不会变成 window 的属性

类型转换

number => string :
String(n)
n + “”

string => number :
Number(s)
parseInt(s)/parseFloat(s)
s - 0

x => bool :
Boolean(x)
!!x

x => string :
String(x)
x.toString()
