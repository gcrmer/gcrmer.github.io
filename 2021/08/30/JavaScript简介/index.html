JavaScript 语言的介绍

JavaScript 是一种高级的、解释型的编程语言，它支持面向对象程式设计，指令式编程，以及函数式编程。在语法上与 Java 有很多的相似性，但是在设计上主要受 Self 语言和 Scheme 语言的影响。

JavaScript 语言的产生

1994年，网景公司（Netscape）发布了 Navigator 浏览器 0.9 版。这个版本的浏览器只能用于浏览，不具备与访问者互动的能力。此时公司需要一种网页脚本语言来实现浏览器与网页的互动，但是选择哪种语言令公司非常犯难。在 1995 年 Sun 公司正式向市场推出 Java 语言后，网景公司决定与其结盟并且在之后做出决策，未来的网页脚本语言必须看上去与 Java 足够相似，但是比 Java 简单。Brendan Eich 被指定为这种 “简化版 Java语言” 的设计师,但是他对 Java 一点兴趣也没有，为了完成公司安排的任务仅仅用了 10 天就完成了 JavaScript 语言的设计。

JavaScript 语言的特点

  • 是一种解释性脚本语言。
  • 主要用来向 HTML 页面添加交互行为。
  • 可以直接嵌入 HTML 页面，但写成单独的 js 文件有利于结构和行为的分离。

JavaScript 语言的用处

  • 嵌入动态文本于 HTML 页面
  • 对浏览器事件作出响应
  • 读写 HTML 元素
  • 在数据被提交到服务器之前验证数据
  • 检测访客的浏览器信息
  • 控制 cookie，包括创建和修改等

JavaScript 语言的标准

JavaScript 已经被 Netscape 公司提交给 ECMA 制定为标准，称之为 ECMAScript，标准编号 ECMA-262。最新版为 ECMAScript 6 。

JavaScript 语言的缺陷

  1. 不适合开发大型程序
  2. 标准库非常小
  3. 全局变量难以控制
  4. 自动插入行尾分号
