a 标签的用法

  1. 作用： a 标签主要做超链接，可以跳转外部页面、内部锚点、邮箱以及电话

  2. 它有以下几种属性

  • href 写一个网址进行引用
  • target 确定在哪个窗口打开超链接
  • download 理论上是下载网页，实际上浏览器可能不支持
  • rel =noopener 防止 Bug
  1. a 的 href 取值有以下几种

  • 网址

    以 “https” “http” 或 “/” 开头 + 域名 三种类型，用“/”+域名”的方式比较保险，不用考虑网页是按照 https 还是 http 的方式请求。

  • 路径

    /a/b/c 或者 a/b/c， 注意当 http 服务开启后，“/”不再是表示计算机本地的根目录，而是表示当前服务的根目录。

  • 伪协议

    • Javascript 代码，点击链接后就会执行
    • tel:手机号
    • mailto: 邮箱

  • id

    href=“#xxx”跳转到指定标签

  1. a 的 target 取值
  • 内置名字

​ _blank 在空白窗口打开

​ _top 在最顶级窗口打开

​ _parent 在当前连接所在 iframe 的上一层打开

​ _self 默认在当前窗口打开

  • 自己命名 windows 的 name

​ 当写 name 的时候定义一个字符串，target = “xxx” 可以实现在同一个窗口打开不同的页面
​ target=“xxx”意思是如果有窗口 xxx 就用 xxx 窗口打开，如果没有 xxx 窗口 就创建一个 xxx 窗口打开

img标签的用法

  1. 作用 : 发出 get 请求，展示图片

  2. 属性：

    • alt （写加载失败后提示用户的话）
    • height（图高 ）width（图宽）只写其中一个另一个会自适应，同时写会导致图片变形
    • src

  3. 事件：onload/onerror 监听图片是否加载成功

  4. 响应式：max-width：100% （保证在手机上也能正常使用

table标签的用法

  1. 作用: table 标签主要是用于做表格
  2. 相关标签
  • thead 表头
  • tbody 内容
  • tfoot 表尾
  • tr 表的一行
  • td 表的数据
  1. 相关样式

  • table-layout auto 宽度根据内容决定 fixed 宽度尽量平均

  • border-collapse 合并边框

  • border-space 调整边框距离
