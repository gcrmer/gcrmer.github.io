Canvas与SVG的区别

2022/06/23

什么是 Canvas

Canvas 基于 HTML canvas 标签，主要是用笔刷来绘制 2D 图形的，它的图像单位是像素。

什么是 SVG

SVG 基于 XML 元素，主要是用标签来绘制不规则矢量图的，可通过 JS 给元素添加点击事件。

Canvas 与 SVG 的相同点

都是主要用来画 2D 图形的。

Canvas 与 SVG 的不同点

1.Canvas 画的是位图，SVG 画的是矢量图，在放大图形后，由于 Canvas 依赖分辨率，会造成失真，而 SVG 不会失真，它比较适合做地图应用。

2.SVG 节点过多时渲染慢，Canvas 性能更好一点，但写法更复杂。所以 Canvas 适合开发图像密集型的游戏，频繁的重新绘制图像，而 SVG 不适合开发游戏应用。

3.SVG 支持分层和事件，Canvas 不支持，但是可以用库实现。

4.SVG 支持事件处理器，而 Canvas 不支持事件处理器。

5.SVG 图像中的文字可以灵活的进行编辑，而 Canvas 的文本渲染能力较弱。
