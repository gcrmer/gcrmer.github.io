运动代码的实现

1.使用 string 保存要显示的结果
使用 string2 缓存要显示到屏幕上的结果

2.构造 step()函数: 作用就是将 string 先缓存到 string2 里，然后再将内容显示到页面上。

注：本来代码中应该这样写 style.innerHTML= string2，意思是我们在写 HTML 的时候同时将 CSS 也写进去，但是会有个问题就是 CSS 中不能出现 HTML 标签的，比如&nbsp，所以这里改用 style.innerHTML= string.substring(0,n);它的作用是显示第 0 个到第 n 个字符串，意思就是 string 里有什么就显示什么，有空格就显示空格。
